## 🚀 Project Overview
Somaliguard is a professional security dashboard designed to detect fraud in Somali mobile money services. It uses a multi-layered architecture to process data through a machine learning model and display real-time results.

## 🛠️ The Real Tech Stack
- **Frontend:** Next.js & Tailwind CSS (The security dashboard interface).
- **Backend API:** Node.js (The engine that processes requests and connects the UI to the AI).
- **AI Engine:** Python & FastAPI (Where the machine learning model lives).
- **Machine Learning:** Scikit-learn (Used to train the fraud detection logic).
- **Deployment:** Render (Hosting the static frontend and the web services).
