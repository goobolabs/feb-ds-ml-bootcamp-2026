# 🎫 IT Support Ticket Priority Prediction

### 🧠 Project Overview

This project uses **Machine Learning** to automatically classify IT support tickets into three categories: **High, Medium, and Low** priority.

By reading the "Body" of a ticket, the AI helps IT departments in organizations to fix critical issues first, reducing manual sorting time by 80%.
