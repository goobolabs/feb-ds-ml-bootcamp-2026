# Disease Predictor | AI-Based Medical Diagnosis Assistant

## Project Title
AI-Powered Disease Prediction System Using Machine Learning

## Short Description
An end-to-end Machine Learning system that predicts diseases based on patient symptoms with 94%+ accuracy. The system analyzes 123 symptoms to classify 53 different diseases including cancers, heart disease, diabetes, HIV/AIDS, and respiratory conditions. It compares Random Forest (93.87% accuracy) and Naive Bayes (94.46% accuracy) algorithms to provide top-K ranked predictions with confidence scores. The trained models are deployed via a Flask REST API with an interactive web dashboard featuring real-time symptom search and multi-disease prediction capabilities.
