Project Description
​This is a Full-Stack Machine Learning project designed to predict a student's final academic grade based on various factors such as study hours, attendance rate, and previous scores. The system utilizes a Flask API for the backend and a Next.js interface for the frontend. It compares two algorithms, Linear Regression and Random Forest, to provide the most accurate predictions.
​Key Features
​Dataset: 1,000 student records with 4 primary features.
​Models: Comparison between Linear Regression and Random Forest.
​Deployment: Fully functional API with a /predict endpoint.
​Frontend: A modern, responsive UI for real-time predictions.
