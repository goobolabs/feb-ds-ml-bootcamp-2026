# 🫀 Cardiovascular Disease Predictor

## 🧠 Project Overview
This project uses Machine Learning to automatically predict cardiovascular health risks based on medical patient data. 

By analyzing key patient metrics (such as age, blood pressure, and cholesterol level), the AI helps healthcare professionals evaluate potential heart conditions effectively and accurately.

Accuracy: The system is around 73-75% accurate, which is a good start for this type of medical data.

Tech Stack: Python, FastAPI, Jinja2 Templates (HTML/CSS), and Scikit-learn.
